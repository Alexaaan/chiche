# Design Plan for Cosmic Candy Web App

## Overview
Modern showcase web app with colorful, dreamy, premium visual identity inspired by cosmic universe, clouds, stars, and candies.

## Color Palette
- Primary gradients: violet (#8B5CF6) → rose (#EC4899) → orange (#F97316) → yellow (#EAB308)
- Accents: luminous whites, pastels
- Background: soft cosmic gradient with texture

## Typography
- Titles: 'Fredoka One' (rounded, fun, expressive)
- Body text: 'Poppins' (modern sans-serif)
- Hierarchy: Clear distinction between headings and content

## Layout Structure

### Header
- Large immersive header
- Cosmic background with gradient and star particles
- Centered logo: rounded, cartoon style with glow effect
- Simple navigation menu (Accueil, Produits, À propos, Contact)

### Products Section
- Responsive grid: 3 columns on desktop, 2 on tablet, 1 on mobile
- Product cards:
  - Rounded corners
  - Soft shadow
  - Prominent product image
  - Product name in fun typography
  - Secondary info (price, stock status)
  - Rounded action button
  - Hover effects: elevation and glow

## Interactive Effects
- Smooth transitions (200-300ms)
- Glow effects on elements
- Hover: card lift and glow
- Responsive design for all devices

## Implementation Notes
- Use CSS gradients for backgrounds
- Add Google Fonts imports
- Ensure accessibility and performance
- Mobile-first responsive design
- Compatible with Netlify (static site hosting)
